##0969DA SQL-project
Rockbuster Stealth SQL Analysis – Project Overview
# Introduction
This project is part of the CareerFoundry Data Analytics program and focuses on analyzing the Rockbuster Stealth LLC movie rental database. Rockbuster Stealth is a global film rental company preparing to launch an online streaming platform to remain competitive against major players such as Netflix and Amazon Prime.

The goal of this project is to explore the company’s relational database using SQL, extract meaningful insights, and support strategic decision‑making for the upcoming digital launch.

# Project Context
The dataset contains information about Rockbuster’s film inventory, customers, payments, and rental activity.
The analysis involves:

Loading the dataset into a PostgreSQL relational database

Understanding the schema, relationships, and key data points

Writing SQL queries of varying complexity

Producing insights relevant to business stakeholders

Presenting results in a clear and accessible format

This project demonstrates practical SQL skills applicable across industries.

# Key Business Questions
The analysis addresses several strategic questions defined by the Rockbuster management team:

Which movies generated the highest and lowest revenue

What is the average rental duration across all films

In which countries Rockbuster customers are located

Where high–lifetime value customers are based

Whether sales performance varies across geographic regions

These questions require a combination of joins, aggregations, subqueries, and CTEs.
